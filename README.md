## About this project

This is a front end app fetching data from the PokeAPI. The main objective of this project is to develop a client app that fetches data from an API and renders it for the user. It is developed with Next js.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

The pages auto-updates as you edit the file.
